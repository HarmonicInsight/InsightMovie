# クイックスタートガイド

InsightMovieを使い始めるための最短手順

## 📦 インストール（5分）

### 1. インストーラーをダウンロード

[Releases](https://github.com/yourusername/insightmovie/releases)から最新版をダウンロード

### 2. インストール実行

1. `InsightMovie-Setup-1.0.0.exe` をダブルクリック
2. 「次へ」を数回クリック
3. ✅ **重要**: 「VOICEVOX自動セットアップ」に**必ずチェック**
4. インストール完了まで待つ（数分かかります）

> **注意**: VOICEVOX自動セットアップはインターネット接続が必要です

## 🚀 初回起動（3分）

### 1. アプリを起動

スタートメニューまたはデスクトップから「InsightMovie」をクリック

### 2. セットアップウィザード

自動的に表示されます。「次へ」を押すだけ！

1. ようこそ画面 → 「次へ」
2. エンジン検出 → 自動で検出します → 「次へ」
3. 話者設定 → 「青山流星」が自動設定 → 「次へ」
4. 完了 → 「完了」

## 🎙️ 音声生成（1分）

### 簡単3ステップ

1. **テキスト入力**
   ```
   こんにちは、InsightMovieです。
   ```

2. **生成ボタンをクリック**
   「音声を生成」ボタンをクリック

3. **保存**
   「WAVファイルとして保存」で保存

完成！🎉

## ❓ トラブルが起きたら

### エンジンが見つからない

1. ウィザードで「エンジンを起動」ボタンをクリック
2. 数秒待つ
3. 「再検索」をクリック

### それでもダメなら

1. 「手動設定」をクリック
2. 以下のパスを探す:
   ```
   C:\Users\[ユーザー名]\AppData\Local\InsightMovie\voicevox\run.exe
   ```
3. run.exeを選択して「開く」

## 💡 ヒント

### 音声の調整

現在のバージョンでは「青山流星」のみ使用可能です。
今後のバージョンで話者変更機能を追加予定！

### オフライン使用

一度セットアップが完了すれば、インターネット接続なしで使用できます。

### ファイル保存場所

デフォルトではホームフォルダに保存されます。
好きな場所を選択できます。
